# CanDccBooster

Bibliothèque Discovery 2026 pour booster DRV8874 avec RailCom HF.

## Fonctionnalités
- Réception DCC via CAN (ID 0x100, 0x101)
- Application DCC sur DRV8874
- Gestion du cutout RailCom
- Lecture RailCom HF
- Calibration automatique des seuils
